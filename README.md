# Sorting Visualizer ✨

Hello **eng lolo**! This is a simple, clean, and modular web application for visualizing how different sorting algorithms work.

## Features
- **Bubble Sort**: Compares adjacent elements and swaps them if they are in the wrong order.
- **Selection Sort**: Finds the minimum element and moves it to the beginning.
- **Insertion Sort**: Builds the sorted array by inserting each element into its correct position.
- **Customizable Speed**: Adjust how fast the visualization runs.
- **New Array**: Generate a fresh set of random bars anytime.

## Tech Stack
- **HTML5**: Semantics and structure.
- **CSS3**: Clean, feminine design with soft pastel colors (no gradients).
- **Vanilla JavaScript**: Modular logic using ES6+ (`let`, `const`, `async/await`).

## Project Structure
- `index.html`: The main user interface.
- `css/style.css`: All the beautiful styling.
- `js/visualizer.js`: Handles bar rendering and UI updates.
- `js/sorting-algorithms.js`: Contains the mathematical logic for each sort.
- `js/app.js`: The main controller that connects everything.

## How to Run
- Simply open the `index.html` file in any modern web browser!
