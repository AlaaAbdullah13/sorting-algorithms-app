## 🎯 Features

| Feature | Description |
|---------|-------------|
| 🫧 **5 Sorting Algorithms** | Bubble, Selection, Insertion, Merge, and Quick Sort |
| ▶️⏹️🔄 **Playback Controls** | Start, Stop mid-sort, and Reset to original array |
| ⏩⏪ **Speed Controls** | Speed Up / Slow Down buttons + adjustable slider |
| 📏 **Array Size Slider** | Dynamically resize the array from 10 to 80 bars |
| 📊 **Live Statistics** | Real-time comparison count, swap count, and elapsed time |
| 🌓 **Dark/Light Mode** | Toggle between a soft lavender light theme and neon purple dark theme |
| 🎨 **Cartoon Design** | Bubbly "Fredoka" font, thick borders, bouncy animations, bold purple palette |
| 📱 **Responsive** | Works beautifully on desktop, tablet, and mobile |

## 🧮 Algorithms

| Algorithm | Time (Best) | Time (Average) | Time (Worst) | Space | Stable |
|-----------|:-----------:|:--------------:|:------------:|:-----:|:------:|
| 🫧 Bubble Sort | O(n) | O(n²) | O(n²) | O(1) | ✅ |
| 👆 Selection Sort | O(n²) | O(n²) | O(n²) | O(1) | ❌ |
| 📥 Insertion Sort | O(n) | O(n²) | O(n²) | O(1) | ✅ |
| 🔀 Merge Sort | O(n log n) | O(n log n) | O(n log n) | O(n) | ✅ |
| ⚡ Quick Sort | O(n log n) | O(n log n) | O(n²) | O(log n) | ❌ |

## 🎨 Design Philosophy

This project uses a **purple cartoon aesthetic** inspired by playful, bubbly design:

- 💜 **Purple palette** — Deep violets, soft lavenders, and neon accents
- 🖋️ **Fredoka font** — A rounded, cartoon-style Google Font
- 🫧 **Thick borders & big shadows** — Bold, tactile button design
- 🎪 **Bouncy animations** — Bars bounce in, buttons spring on hover
- 🌈 **Color-coded states** — Cyan for comparing, pink for swapping, purple for sorted

## 🚀 How to Run

1. **Clone the repository:**
   ```bash
   git clone https://github.com/AlaaAbdullah13/sorting-algorithms-app.git
   ```
2. **Open `index.html`** in any modern browser — that's it!
   ```bash
   cd sorting-algorithms-app
   open index.html    # macOS
   xdg-open index.html # Linux
   ```

> No build tools, no dependencies, no npm install. Just open and enjoy! 🎉

## 🎮 Controls Guide

| Button | Action |
|--------|--------|
| 🔀 **New Array** | Generate a fresh random array |
| ▶️ **Start** | Begin sorting with the selected algorithm |
| ⏹️ **Stop** | Interrupt the sorting process mid-sort |
| 🔄 **Reset** | Restore the array to its pre-sort state |
| ⏩ **Speed Up** | Increase sorting speed by 10 |
| ⏪ **Slow Down** | Decrease sorting speed by 10 |
| 🌓 **Theme** | Toggle dark/light mode |
